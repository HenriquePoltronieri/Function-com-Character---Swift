# 🔤 Function com Character - Swift

Atividade bônus desenvolvida em **Swift** utilizando **Xcode Playground**, com foco no uso de funções e dicionários para contagem de caracteres.

O projeto implementa a função `contar`, que recebe uma `String` e retorna um dicionário `[Character: Int]` com a frequência de cada caractere no texto.

---

## 📤 Saída esperada

```
["a": 3, "b": 1, "c": 1, "t": 1, "e": 1]
```

> A ordem das chaves pode variar, pois dicionários em Swift não são ordenados.

---

## 🛠️ Tecnologia

- [Swift](https://www.swift.org/)
- Xcode Playground
